# Paleta de Cores - Landing Page Advocacia

## 🎨 Cores Aplicadas

### Cores Principais

**Preto e Cinzas:**
- `#000000` - Preto puro (fundo do hero, footer)
- `#1a1a1a` - Preto suave (cor primária, textos importantes)
- `#2b2b2b` - Cinza escuro (textos, ícones)
- `#4a4a4a` - Cinza médio (elementos secundários)
- `#6b6b6b` - Cinza claro (textos secundários)

**Dourado:**
- `#c9a961` - Dourado principal (botões, destaques, hover)
- `#b89751` - Dourado hover (estado hover de botões)

**Cores Neutras:**
- `#ffffff` - Branco (fundo de cards, textos em fundo escuro)
- `#f5f5f5` - Cinza muito claro (fundo de seções alternadas)
- `#e8e8e8` - Bordas suaves

## 📍 Onde as Cores são Aplicadas

### Preto (#000000 / #1a1a1a)
- ✓ Fundo do Hero Section (gradiente)
- ✓ Fundo do Footer
- ✓ Títulos principais
- ✓ Ícones de áreas de atuação
- ✓ Ícones de contato
- ✓ Menu de navegação

### Dourado (#c9a961)
- ✓ Botões de Call-to-Action
- ✓ Ícones (dentro dos círculos escuros)
- ✓ Links em hover
- ✓ Estrelas de avaliação
- ✓ Títulos de seções no footer
- ✓ Bordas em hover dos cards
- ✓ Destaques e elementos interativos

### Cinza (tons variados)
- ✓ Textos secundários
- ✓ Descrições
- ✓ Bordas de cards
- ✓ Backgrounds de elementos

## 🎯 Conceito Visual

A paleta foi escolhida para transmitir:

- **Seriedade** - Tons escuros de preto e cinza
- **Profissionalismo** - Contraste equilibrado
- **Elegância** - Dourado como destaque refinado
- **Confiança** - Cores sólidas e neutras
- **Sofisticação** - Gradientes sutis em preto

## 💡 Dicas de Uso

### Para manter a consistência visual:

1. **Textos Importantes**: Use preto (#1a1a1a)
2. **Textos Secundários**: Use cinza (#6b6b6b)
3. **Destaques**: Use dourado (#c9a961)
4. **Botões Principais**: Fundo dourado com texto preto
5. **Hover Effects**: Sempre aplique o dourado
6. **Ícones**: Fundo preto/cinza escuro com ícone dourado

## 🔧 Como Alterar as Cores

Para mudar as cores, edite o arquivo `style.css` na seção `:root` (linhas 8-17):

```css
:root {
    --primary-color: #1a1a1a;      /* Preto principal */
    --secondary-color: #c9a961;    /* Dourado */
    --dark-color: #000000;         /* Preto puro */
    --light-color: #f5f5f5;        /* Cinza clarinho */
    --white: #ffffff;              /* Branco */
    --text-color: #2b2b2b;         /* Cinza texto */
    --gray: #6b6b6b;               /* Cinza médio */
    --medium-gray: #4a4a4a;        /* Cinza escuro */
}
```

## 🎨 Variações Alternativas (Opcionais)

Se quiser ajustar o tom do dourado:

- **Mais claro**: `#d4bf8e` (dourado champagne)
- **Mais escuro**: `#a18545` (dourado bronze)
- **Mais vibrante**: `#d4af37` (dourado metálico)

- Destaques visuais em dourado sobre fundos escuros
