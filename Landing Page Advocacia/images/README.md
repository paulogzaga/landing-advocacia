# Instruções para Adicionar as Logos

Para completar a landing page, você precisa adicionar as imagens das logos nesta pasta:

## Imagens Necessárias:

1. **logo-black.png** - Logo com fundo preto (primeira imagem que você enviou)
   - Use esta imagem no header (navbar)
   - Salve como: `logo-black.png`

2. **logo-white.png** - Logo com fundo branco (segunda imagem que você enviou)
   - Use esta imagem na seção "Sobre" e no footer
   - Salve como: `logo-white.png`

## Como Adicionar:

1. Salve as duas imagens que você enviou nesta pasta `images/`
2. Renomeie-as conforme indicado acima
3. As imagens já estão configuradas no HTML e CSS

## Dica:

- A logo no header (navbar) ficará melhor com a versão em fundo branco, pois o navbar é branco
- A logo na seção "Sobre" ficará melhor com a versão em fundo branco, pois a seção tem fundo preto
- Você pode inverter se preferir, basta trocar os nomes dos arquivos

---

Após adicionar as logos, atualize a página no navegador para ver o resultado final!
